Exercise 19 - Customization with Sass

Exercise 19.1

Step 1: Setup Your Project

    mkdir bootstrap-sass-project
    cd bootstrap-sass-project
    npm init -y
    npm install bootstrap sass

Step 2: Create Your SCSS Files

_variables.scss (override defaults)
  // _variables.scss
  $primary: #ff5722;
  $border-radius: 1rem;
custom.scss (import variables + Bootstrap)
  // custom.scss
  @import "variables";
  @import "../node_modules/bootstrap/scss/bootstrap";

Step 3: Compile Sass

  npx sass scss/custom.scss dist/custom.css

Or add to package.json scripts:

  "scripts": {
    "build": "sass scss/custom.scss dist/custom.css"
  }

Step 4: Use Customized Bootstrap CSS

Example HTML snippet:

  <link rel="stylesheet" href="dist/custom.css" />
  
  <button class="btn btn-primary rounded">Rounded Button</button>
